Detailed Description

PHP code to generate HTML output of a topic

The basic call is topic.php$key=### but .htaccess contains a rewrite of the form ^topics?/([0-9]+).html?$ topic.php?key=$1 which allows

  • /topic/###.htm
  • /topics/###.htm
  • /topic/###.html
  • /topics/###.html

This routine checks that the $_GET variable is valid and then pulls the topic, all associated blogs & volumes and comments and ends with a call to the Twig templage generator, a file topic.twig
